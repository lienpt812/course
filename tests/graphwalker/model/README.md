# Graph JSON Models

Thư mục này dùng để lưu các file JSON mô tả graph phục vụ thiết kế test MBT.

Quy ước gợi ý:

- Mỗi luồng nghiệp vụ một file JSON riêng.
- Tên file theo mẫu: `<flow-name>.json`.
- Ưu tiên lưu cùng khái niệm với model `.graphml` tương ứng.

Ví dụ:

- `course-registration-flow.json`
- `learning-progress-flow.json`
